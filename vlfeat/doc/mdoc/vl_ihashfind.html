SEL = VL_IHASHFIND(ID, NEXT, K, X) returns a vector SEL of the entires in the hash table ID,NEXT,K corresponding to the labels stored as columns of X.

The format is the same as for the functino VL_IHASHSUM().

See also: VL_IHASHSUM().
